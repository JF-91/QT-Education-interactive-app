# Aplicación Educativa Interactiva
Un lector diseñado para libros de texto que integra contenido multimedia.

### Características: 
Leer un PDF que contiene "zonas clicables". Al hacer clic en una imagen o gráfico, se abre un panel lateral con video explicativo, un formulario de cuestionario interactivo o una definición contextual.
Tecnología clave:w Qt WebEngine incrustado para manejar contenido HTML/JS dentro del documento, o un sistema propio de "hotspots" mapeados sobre las páginas del PDF renderizado.
